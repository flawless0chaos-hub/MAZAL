# 🎰 Family Slot Arena - מדריך הפעלה והעלאה לאינטרנט

## 📋 דרישות מערכת

לפני שמתחילים, תוודא שיש לך מותקן:
- **Node.js** גרסה 18 ומעלה
  - להורדה: https://nodejs.org/
  - בחר ב-"LTS" (Long Term Support)

---

## 🚀 שלב 1: הפעלה מקומית במחשב שלך

### 1.1 פתח את התיקייה
```bash
cd family-slot-arena
```

### 1.2 התקן את התלויות
```bash
npm install
```
(זה לוקח בערך 1-2 דקות)

### 1.3 הפעל את השרת המקומי
```bash
npm run dev
```

### 1.4 פתח בדפדפן
פתח את הכתובת: **http://localhost:3000**

🎉 **עכשיו המשחק עובד במחשב שלך!**

---

## 🌍 שלב 2: העלאה לאינטרנט (חינם!)

אני ממליץ על **Vercel** - זה חינם לחלוטין ומאוד פשוט!

### אופציה A: דרך הממשק (הכי קל!)

1. **צור חשבון בחינם:**
   - לך ל: https://vercel.com/signup
   - התחבר עם GitHub / Google / Email

2. **העלה את הפרויקט:**
   - לחץ על "Add New..." → "Project"
   - גרור את התיקייה `family-slot-arena` או העלה כ-ZIP
   - לחץ "Deploy"

3. **זהו! תוך 2 דקות תקבל קישור:**
   - משהו כמו: `https://family-slot-arena.vercel.app`
   - שלח את הקישור למשפחה! 🎊

---

### אופציה B: דרך שורת הפקודה (למתקדמים)

```bash
# התקן Vercel CLI
npm i -g vercel

# התחבר
vercel login

# העלה את הפרויקט
vercel --prod
```

## 🔧 פתרון בעיות נפוצות

### "npm: command not found"
➡️ צריך להתקין Node.js מ: https://nodejs.org/

### "Port 3000 is already in use"
➡️ הפעל בפורט אחר:
```bash
PORT=3001 npm run dev
```

### המשחק לא נטען
➡️ בדוק את הקונסול בדפדפן (F12)
➡️ נקה Cache (Ctrl+Shift+Delete)

### העלאה ל-Vercel נכשלה
➡️ בדוק שיש לך אינטרנט
➡️ התחבר מחדש: `vercel login`
