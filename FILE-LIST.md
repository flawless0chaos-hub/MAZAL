# 📂 תוכן התיקייה - Family Slot Arena

## 🎮 קבצי הפעלה ראשיים

### START.bat (Windows)
- **מה זה:** סקריפט הפעלה אוטומטית ל-Windows
- **איך משתמשים:** לחיצה כפולה
- **מה הוא עושה:**

  2. מפעיל את השרת (npm run dev)
  3. פותח את המשחק בדפדפן אוטומטית

### START.sh (Mac / Linux)
- **מה זה:** סקריפט הפעלה אוטומטית ל-Mac/Linux
- **איך משתמשים:** לחיצה כפולה (או `./START.sh` בטרמינל)
- **מה הוא עושה:** 
  1. מתקין את כל התלויות (npm install)
  2. מפעיל את השרת (npm run dev)
  3. פותח את המשחק בדפדפן אוטומטית

- **מה בפנים:**

  - טבלת פתרון בעיות
  - טקסט לשליחה

### QUICK-START.txt
- **מה זה:** מדריך מהיר בטקסט רגיל (ללא HTML)

- **מה בפנים:**
  - 3 שלבים פשוטים
  - פתרון בעיות
  - טקסט לשליחה

### README.md
- **מה זה:** מדריך מלא ומפורט בפורמט Markdown

- **למי זה מתאים:** למי שרוצה את כל הפרטים
- **מה בפנים:**

  - פתרון בעיות מקיף

## 📁 תיקיות

### app/
- **מה זה:** תיקייה ריקה (מיועדת ל-Next.js 13+)
- **למה היא קיימת:** Next.js מצפה לה
- **האם צריך משהו בפנים:** לא, היא יכולה להישאר ריקה

### public/
- **מה זה:** תיקייה לקבצים סטטיים (תמונות, פונטים, וכו')
- **מה לשים בפנים:** אפשר להוסיף לוגו, תמונות, אייקונים
- **איך לגשת:** מהקוד ב-`/filename.png` (עם / בהתחלה)

### pages/
- **מה זה:** כל העמודים של האפליקציה
- **מה בפנים עכשיו:**
  - `index.js` - עמוד הבית (המשחק)
  - `_app.js` - עטיפת האפליקציה
